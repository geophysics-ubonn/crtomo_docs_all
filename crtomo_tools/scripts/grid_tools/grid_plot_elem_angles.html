grid_plot_elem_angles module¶

Plot a nice histogram of element angles to plot_element_angles.jpg. The script expects an elem.dat file in the present working directory.

Examples

>>> ls -1
elec.dat
elem.dat
>>> grid_plot_elem_angles
>>> ls -1
elec.dat
elem.dat
plot_element_angles.jpg
