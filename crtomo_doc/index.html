Introduction¶

The CRTomo-ecosystem is maintained in four repositories:

  • crtomo: the CRMod/CRTomo/CutMcK programs, written in the Fortran programming language
  • crtomo_tools: user space tools that help with the various tasked associated with using CRMod/CRTomo, such as grid creation, analysis of modeling/inversion results, and plotting of various results. Mostly written in the Python programming language.
  • crtomo_doc: this documentation. The documentation should provide links to external literature, but should be self-contained in the way that novice users should be able to install, configure, and use CRMod/CRTomo only by reading this documentation.
  • crtomo_test: Test and use cases for CRTomo. This is part automatic test suite, part manual storage location for ‘interesting’ use cases.

We aim at full usability on both Linux and Windows operating systems.
